# Generated QR Codes

This directory contains QR codes for the sample medical IDs used in the Medical QR Scanner Web App.

## QR Code Types

For each sample medical ID, two QR codes have been generated:

1. **Raw ID QR Codes** - Contains just the medical ID (e.g., `MEDQR-0001`)
2. **URL QR Codes** - Contains the full URL format (e.g., `https://medical-id.example.com/p/MEDQR-0001`)

## Usage

These QR codes can be used to test the Medical QR Scanner Web App:

1. Open the web app in your browser
2. Use the "Start Camera Scan" feature to scan the QR codes with your device's camera
3. Alternatively, use the "Select QR Image" feature to upload the QR code images directly

Both the raw ID and URL formats will be correctly parsed by the application.
